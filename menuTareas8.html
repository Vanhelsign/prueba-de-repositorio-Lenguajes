Tarea #8. R

Simbolos que utiliza R


1-Operadores matematicos


  • `+` (Suma).
  • `-` (Resta).
  • `*` (Multiplicación).
  • `/` (División).
  • `%%` (Modulo).
  • `^` ó `**` (potencia).
  • `%/%` (División entera).

2-Operadores de comparación


  • `==` (Igual a).
  • `!=` (Diferente de).
  • `<` (Menor que).
  • `>` (Mayor que).
  • `<=` (Menor o igual que).
  • `>=` (Mayor o igual que).

3-Operadores de concatenación


  • ` c() ` (Concatenación de vectores ).
  • ` paste() ` (Concatenación de cadenas de texto).
  • ` rbind() ` (Unión de marcos de datos por filas).
  • ` cbind() ` (Unión de marcos de datos por columnas).

4-Simbolos de control de flujo


  • ` if, else if, if else ` (Estructuras condicionales).
  • ` for ` (Para bucles tipo for).
  • ` while ` (Para bucles while).
  • ` repeat ` (Para bucles infinitos).

5-Operadores logicos


  • `& ó && ` (Puerta logica "Y").
  • `| ó || ` (Puerta logica "o").
  • ` ! ` (Negación).

6-Comentarios


  • ` # ` (Para comentarios de una sola línea).


Sintaxis que utiliza R


  • Variables: Para asignar valores a variables en R, se utiliza el operador de asignación "<-" o el signo igual "=" , ademas de que los nombres de las variables son sensibles a mayúsculas y minúsculas.

    Ejemplo: mi_variable <- 42
    -------------otra_variable = "Hola, mundo"

  • Comentarios: Se utilizan para añadir explicaciones de una o varias lineas de codigo, esto utilizando el simbolo #.

    Ejemplo: #Este es un comentario en R.

  • Estructuras de control: R admite estructuras de control como if, else, for, while, y repeat para controlar el flujo de un programa.

    Ejemplo:

    if (condición){
    --- --Condición si la condición es verdadera-----}
    --else{
    --- --Codigo si la condición es falsa-----}

  • Funciones: Se Puede definir funciones en R utilizando la palabra clave "function", las cuales esas funciones toman argumentos y pueden devolver resultados.

    Ejemplo:

    mi_funcion <- function(x, y) {
    --resultado <- x + y
    --return(resultado) }

  • Vectores: R es especialmente eficiente en el trabajo con vectores, lo que para crear un vector, se puede hacer uso de la función c() o simplemente enumerar los elementos entre corchetes [].

    Ejemplo: mi_vector <- c(1, 2, 3, 4, 5)
    ------------mi_vector2 <- 1:5

  • Indezación:Se puede acceder a elementos en vectores, matrices y listas utilizando corchetes [ ].

    Ejemplo: mi_vector[1] #Accede al primer elemento del vector

  • Importación de bibliotecas: Mediante el uso de las funcion "library" es que se puede agregar una biblioteca externa a las librerias establecidas en R.

    Ejemplo: library(dplyr) # Cargar el paquete dplyr

Aplicación a descargar para editar y ejecutar


  • R Studio: Se trata de un entrono de desarrollo integrado de codigo abierto que ofrece a los programadores de R acceso a diversas herramientas y funcionalidades; Entre las herramientas que incluye este IDE esta la visualización de datos, la depuracioón de codigo, la gestion de proyectos, la compatibilidad con sistemas de control de versiones, etc.

  • Visual Studio Code: Es uno de los editores mas populares de código R, dado a que con la extensión que este proporciona para la creación de programación en R, es que permite la disponibilidad de muchas funciones a utilizar, como el resaltado de sintaxis, la depuración del codigo, integración con GIT, autocompletado, etc.

  • Eclipse: El entorno de desarrollo Eclipse, proporciona al desarrollo del codigo unas cuantas ventajas a comparación de otros, la cual es su extnsa documentación del lenguaje que tienen, en donde explica y detalla sobre las funciones y paquetes que se pueden utilizar para el desarrollo del lenguaje R.

¿A que tipo de paradigma pertenece?


- El lenguaje de programación R, es un lenguaje interpretado centrado principalmente en el paradigma declarativo procedimental estructurado, asi tambien como el de la programación orientada a objetos y programación funcional.


Esqueleto y ejemplo de programa escrito en R- Serie de fibonacci



    fib <- function(N) {
        if (N == 0) {
          return(0)
        } else if (N == 1) {
          return(1)
        } else {
          return(fib(N - 1) + fib(N - 2))   
        }
      }
                  
      n <- 10
      for (i in 0:(n - 1)) {
        print(fib(i))
      }
