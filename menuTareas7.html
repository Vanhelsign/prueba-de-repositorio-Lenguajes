Tarea #7. Haskell

Simbolos que utiliza Haskell


1-Operadores matematicos


  • `+` (Suma).
  • `-` (Resta).
  • `*` (Multiplicación).
  • `/` (División).
  • `sqrt` (Raiz cuadrada).
  • `^` (potencia).

2-Operadores de comparación


  • `==` (Igual a).
  • `!=` (Diferente de).
  • `<` (Menor que).
  • `>` (Mayor que).
  • `<=` (Menor o igual que).
  • `>=` (Mayor o igual que).

3-Simbolos especiales


  • ` \ ` (Se utiliza para definir funciones anonimas).
  • ` $ ` (Se utiliza para aplicar un una función a un argumento).
  • ` -- > ` (Indica la relación entre dos argumentos de entrada y de la función).

4-Simbolos de puntuación


  • ` , ` (Coma, para separar elementos en una lista o argumentos de una declaración).
  • ` :: ` (Dos puntos, se utiliza para especificar el tipo de expresión o función).
  • ` ; ` (Punto, se utiliza para separar declaraciones en un bloque de codigo).

5-Operadores logicos


  • ` && ` (Puerta logica "Y").
  • ` || ` (Puerta logica "o").
  • ` not ` (Negación).

6-Comentarios


  • ` -- ` (Para comentarios de una sola línea).


Sintaxis que utiliza Haskell


  • Variables y asignación: Las variables en Haskell se crean una vez que se le asigna el tipo de dato, esto debido a que las varaibles no son mutables y en cambio se utilizan patrones de coincidencia y funciones para definir valores.

    Ejemplo: nombre :: String
    -------- nombre = "Juan"

  • Comentarios: Se utilizan para añadir explicaciones de una o varias lineas de codigo, esto utilizando el simbolo "--" ó {- -}.

    Ejemplo: --Este es un comentario en Haskell. Ejemplo 2: {- Este es un comentario de varias lineas en Haskell-}

  • Estructuras de control: Con Haskell no es necesario el us ode caracteres para definir bloques de codigo, este se guia mas por la identiación del codigo.

    Ejemplo:

    if condición
    --then
    --- --Condición si la condición es verdadera-----
    --else
    --- --Codigo si la condición es falsa-----

  • Funciones: Para poder crear una función es necesario definirlo con una palabra clave acompañado del simbolo "::" para especificar el tipo de dato y el simbolo "=" para definir la función.

    Ejemplo:

    miFuncion :: Tipo1 -> Tipo2
    miFuncion parametro = -- Código de la función

  • Listas: Para poder crear listas es necesario asignar elementos a una variable haciendo uso de los corchetes "[ ] y para añadir esos elementos a la variable se hace uso de una coma " , ".

    Ejemplo: Mi_lista: [1, 2, 3, 4]

  • Entradas y Salidas: Mediante el uso de las funciones establecidas como `putStrLn` y `getLine` es que se puede hacer uso de la entrada y salida de información .

    Ejemplo:

    main = do
    ----putStrLn "Ingrese su nombre: "
    ----nombre <- getLine
    ----putStrLn ("Hola, " ++ nombre)

  • Importación de bibliotecas: Mediante el uso de las funcion "import" es que se puede agregar una biblioteca externa a las librerias establecidas en Haskell.

    Ejemplo:

    import Data.list (sort)
    resultadoOrdenados = sort [3, 1, 2]

Aplicación a descargar para editar y ejecutar


  • Haskell Lenguaje Server: Es uno de los mejores entornos de desarrollo de este lenguaje, debido a su amplio volumne de herrameintas avanzadas para aquellas personas que lo utilizan para la creación de servicios avanzados, proporcionando veificación de tipos en tiempo real, renombrado seguro, busqueda extensa de documentación, etc.

  • Visual Studio: Es uno de los editores mas populares de código, que con la extensión "Haskell Language Server", proporciona un entorno de desarrollo para Haskell que incluye herramientas como el resaltado de sintaxis, la verificación de tipos, autocompletado, etc.

  • Intellij IDEA: Es un entorno de desarrollo integrado ampliamente utilizado debido por su potente conjunto de herramientas y funcionalidades para el desarrollo de software, como lo son el control de versiones, la integración que tiene con GHCi, el resaltado de sintaxis y de autocompletado.

¿A que tipo de paradigma pertenece?


- Haskell pertenece principalmente al paradigma de la programación funcional; En el que la programación funcional es un enfoque de programación que se centra en tratar los cálculos como evaluaciones de funciones matemáticas y evita los cambios de estado y datos mutables.


Esqueleto y ejemplo de programa escrito en Haskell- Serie de fibonacci



    fib :: Int -> Int
    fib 0 = 0
    fib 1 = 1
    fib n = fib (n - 1) + fib (n - 2)
                
    main :: IO ()
    main = do
        let n = 10
        let fibonacciNumbers = [fib i | i <- [0..n-1]]  
        mapM_ print fibonacciNumbers
