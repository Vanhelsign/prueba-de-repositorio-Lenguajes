Tarea #5. Prolog

Simbolos que utiliza Prolog


1-Operadores de comparación


  • `=` `==` (unificación, para establecer relaciones).
  • `\=` (no unificación, para establecer que dos terminos no son iguales).
  • `<, >, =<, =>` (comparación numerica).

2-Operadores logicos


  • `,` (coma, "y" logico).
  • `;` (punto y coma, "o" logico).
  • `not` (negacion logica).

3-Operadores aritmeticos


  • `+` `-` `*` `**` `/` `//` (operadores basicos de matematicos).
  • `is` (para evaluar expresiones aritmeticas).

4-Predicados y hechos


  • `:-` (signo de dos puntos y guion, se utiliza para definir reglas).
  • `.` (punto, indica el final de una cláusula o hecho).

5-Operador de pertenencia


  • `member` (se utiliza para verificar si un elemento pertenece a una lista).

6-Operador de consulta


  • `?-` (se utiliza para realizar consultas en la consola de prolog).

7-Comentarios


  • `%` (porcentaje, se utiliza para comentarios de una sola línea).

8-Operador de lista vacia


  • `[ ]` (representa una lista vacía).

Sintaxis que utiliza Prolog


  • Hechos: Los hechos en Prolog se expresan mediante cláusulas que afirman relaciones o propiedades, de manera que cada hecho termina con un punto.

    Ejemplo: hombre(juan).

  • Reglas: Las reglas en Prolog se utilizan para definir relaciones lógicas basadas en condiciones y se representan con una cabeza (el consecuente) y un cuerpo (las condiciones) separados por el operador `:-`.

    Ejemplo: padre(X, Y) :- hombre(X), hijo(Y, X).

  • Consultas: Las consultas en Prolog se utilizan para hacer preguntas o buscar soluciones a través de consultas y estas se representan con un operador de interrogación ?.

    Ejemplo: ?- padre(juan, maria).

  • Variables: Las variables en Prolog se inician con una letra mayúscula o un guion bajo y se utilizan para representar valores desconocidos en reglas y consultas.

    Ejemplo: ?- padre(X, Y).

  • Listas: Prolog admite listas que se representan entre corchetes y elementos separados por comas.

    Ejemplo: animales([perro, gato, pájaro]).

Aplicación a descargar para editar y ejecutar


  • Swi-Prolog: Es un entorno de desarrollo dedicado al lenguaje Prolog de codigo abierto, en el cual proporciona un entorno interactivo de consola y un editor de codigo con resaltado de sintaxis con multiples herramientas de analisis y depuracion de un programa.

  • Visual Studio Code: Es un entorno de desarrollo que es multicompatible con distintos lenguajes, esto a base de descargar un paquete de instalación que hara que se proporcione las herramientas para la creación de un programa en el lenguaje Prolog.

  • Eclipse-Prolog: Es una extención del entorno de desarrollo Eclipse que permite el desarrollo y la depuración de programas Prolog.

¿A que tipo de paradigma pertenece?


- Prolog es un lenguaje de programación declarativo que esta basado en el paradigma de la programación logica, en el cual se centra en representar relaciones logicas permitir el razonamiento logico al resolver consultas; Esto dado a que Prolog se compone de hechos y reglas que describen estas relaciones.


Esqueleto y ejemplo de programa escrito en Prolog- Serie de fibonacci



    fib(0, 0).
    fib(1, 1).
    fib(N,Result) :- 
        N > 0,
        N1 is N - 1,
        N2 is N - 2,
        fib(N1, R1),
        fib(N2, R2),
        Result is R1 + R2.
