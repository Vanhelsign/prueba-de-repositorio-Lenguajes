Simbolos que utiliza Python


1-Operadores matematicos


  • `+` (Suma).
  • `-` (Resta).
  • `*` (Multiplicación).
  • `/` (División).
  • `%` (Modulo. para obtener el resto de la división).
  • `**` (potencia).

2-Operadores de comparación


  • `==` (Igual a).
  • `!=` (Diferente de).
  • `<` (Menor que).
  • `>` (Mayor que).
  • `<=` (Menor o igual que).
  • `>=` (Mayor o igual que).

3-Operadores de asignación


  • `=` (Asignación).
  • `+=` (Suma y asignación).
  • `-=` (Resta y asignación).
  • `*=` (Multiplicacion y asignación).
  • `/=` (División y asignación).
  • `%=`(Modulo y asignación).

4-Simbolos de puntuación


  • ` , ` (Coma, para separar elementos en una lista o argumentos de función).
  • ` : ` (Dos puntos, se utiliza en estructuras de control y definiciones de funciones).
  • ` . ` (Punto, para acceder a atributos y métodos de objetos).
  • `( )`(Paréntesis, para invocar funciones y métodos).

5-Operador de identidad


  • ` is ` (Selección de indentidad).
  • ` is not ` (Selección de no identidad).

6-Comentarios


  • ` # ` (Para comentarios de una sola línea).
  • ` ' ' ó " " ` (Para comentarios multilínea).

7-Operadores logicos


  • ` and ` (Puerta logica "Y").
  • ` or ` (Puerta logica "o").
  • ` not ` (Negación).

Sintaxis que utiliza Python


  • Variables y asignación: Las variables se crean una vez que se le asigna un valor, ademas de que no es necesario declarar el tipo de dato a usar.

    Ejemplo: hombre = "juan"
    Ejemplo 2: edad = 23

  • Comentarios: Se utilizan para añadir explicaciones de una o varias lineas de codigo, esto utilizando el simbolo "#" ó las comillas.

    Ejemplo: #Este es un comentario en python

  • Estructuras de control: Se hace uso del simbolo de Dos puntos ( : ), para poder definir bloques de codigo.

    Ejemplo:

    if condición:
    -----#Condición si la condición es verdadera-----
    else:
    -----#Codigo si la condición es falsa-----

  • Funciones: Para poder crear una función es necesario definirlo con una palabra clave acompañado de la palabra "def".

    Ejemplo:

    def mi_funcion (parametro):
    -----#Codigo de la funcion
    -----return resultado

  • Listas: Para poder crear listas es necesario asignar elementos a una variable haciendo uso de los corchetes "[ ] y para añadir esos elementos a la variable se hace uso de una coma " , ".

    Ejemplo: Mi_lista: [1, 2, 3, 4]

  • Entradas y Salidas: Mediante el uso de las funciones establecidas como `inpuit()` e `print()` es que se puede hacer uso de la entrada y salida de información .

    Ejemplo:

    nombre = input("Ingrese su nombre: ")
    print("Hola,", nombre)

  • Importación de bibliotecas: Mediante el uso de las funcion "import" es que se puede agregar una biblioteca externa a las librerias establecidas en Python.

    Ejemplo:

    import math
    resultado = math.sqrt(16)
