Tarea #10. SQL

Simbolos que utiliza SQL


1-Simbolos de control de flujo


  • `SELECT` (Se utiliza para seleccionar datos de una base de datos).
  • `INSERT` (Se utiliza para insertar nuevos registros en una tabla).
  • `UPDATE` (Se utiliza para actualizar registros existentes en una tabla).
  • `DELETE` (Se utiliza para eliminar registros de una tabla).
  • `CREATE TABLE` (Se utiliza para crear una nueva tabla en la base de datos).
  • `ALTER TABLE` (Se utiliza para modificar una tabla existente).
  • `DROP TABLE` (Se utiliza para eliminar una tabla).
  • `CREATE DATABASE` (Se utiliza para crear una nueva base de datos).

2-Operadores de comparación


  • `==` (Igual a).
  • `!=` (Diferente de).
  • `<` (Menor que).
  • `>` (Mayor que).
  • `<=` (Menor o igual que).
  • `>=` (Mayor o igual que).
  • `BETWEEN` (Entre un rango de valores).
  • `LIKE` (Coincidencia de patrones).

3-Operadores de conjuntos


  • ` UNION ` (Combina conjuntos de resultados).
  • ` INTERSECT ` (Devuelve la intersección de conjuntos de resultados).
  • ` EXCEPT ` (Devuelve la diferencia entre conjuntos de resultados).
  • ` IN ` (Comprueba si un valor está en un conjunto de valores).
  • ` ALL ` (Compara un valor con todos los valores en una subconsulta).

4-Operadores de conjuntos


  • ` + ` (Suma).
  • ` - ` (Resta).
  • ` * ` (Multiplicación).
  • ` / ` (División).
  • ` % ` (Modulo).

5-Operadores logicos


  • ` && ` (Puerta logica "Y").
  • ` || ` (Puerta logica "o").
  • ` not ` (Negación).

6-Comentarios


  • ` -- ` (Para comentarios de una sola línea).


Sintaxis que utiliza SQL


  • Select(Consulta de datos): Utilizado para recuperar datos de una tabla, se puede especificar las columnas que deseas seleccionar y aplicar condiciones utilizando la cláusula WHERE.

    Ejemplo:
    SELECT columna1, columna2
    FROM tabla
    WHERE condición

  • INSERT (Inserción de datos): Permite agregar nuevos registros a una tabla, de manera que se debe especificar la tabla y las columnas en las que deseas insertar valores.

    Ejemplo:
    INSERT INTO tabla (columna1, columna2)
    VALUES (valor1, valor2)

  • UPDATE (Actualización de datos): Se utiliza para modificar datos existentes en una tabla y este puede actualizar valores en una o más columnas basándote en una condición.

    Ejemplo:
    UPDATE tabla
    SET columna = nuevo_valor
    WHERE condición

  • DELETE (Eliminación de datos): Permite eliminar registros de una tabla que cumplan con ciertas condiciones especificadas en la cláusula WHERE.

    Ejemplo:
    DELETE FROM tabla
    WHERE condición

  • CREATE TABLE (Creación de tablas): Se utiliza para crear una nueva tabla en la base de datos; Ademas se Debe definir el nombre de la tabla y las columnas que contendrá, junto con sus tipos de datos.

    Ejemplo:
    CREATE TABLE nombre_tabla (
    columna1 tipo_de_dato,
    columna2 tipo_de_dato,
    ...
    )

  • ALTER TABLE (Modificación de tablas): Se utiliza para realizar cambios en la estructura de una tabla existente, como agregar una nueva columna.

    Ejemplo:
    ALTER TABLE nombre_tabla
    ADD columna_nueva tipo_de_dato

  • DROP TABLE (Eliminación de tablas): Elimina una tabla completa y todos sus datos de la base de datos.

    Ejemplo:

    DROP TABLE nombre_tabla

  • Comentarios: Se utilizan para agregar notas o explicaciones en el código SQL.

    Ejemplo:

    -- Esto es un comentario de una línea

¿A que tipo de paradigma pertenece?


- El paradigma de programación de SQL es el imperativo, el cual este esta combinado con instrucciones declarativas con el lenguaje de consulta, dado a que las instrucciones imperativas se usan para ejecutar asignaciónes, operaciones de aritmetica, ejecutar ciclos, etc.

        id SERIAL PRIMARY KEY,
        nombre VARCHAR(50),

        salario NUMERIC,
        departamento VARCHAR(50)
