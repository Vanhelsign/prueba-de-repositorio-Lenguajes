Tarea #9. PHP

Simbolos que utiliza PHP


1-Operadores aritmeticos


  • `+` (Suma).
  • `-` (Resta).
  • `*` (Multiplicación).
  • `/` (División).
  • `%` (Modulo).
  • `**` (potencia).

2-Operadores de asignación


  • `=` (Asignación de variables).
  • `+=` (Suma y asignación).
  • `-=` (Resta y asignación).
  • `*=` (Multiplicación y asignación).
  • `/=` (División y asignación).
  • `%=` (Modulo y asignación).

3-Operadores de comparación


  • `==` (Igual al valor).
  • `===` (Igual al valor y tipo de dato).
  • `!=` (No igual al valor).
  • `!==` (No igual al valor y tipo de dato).
  • `<` (Menor que).
  • `<=` (Menor o igual que).
  • `>` (Mayor que).
  • `>=` (Mayor o igual que).

4-Simbolos de control de flujo


  • ` if, else if, if else ` (Estructuras condicionales).
  • ` for ` (Para bucles tipo for).
  • ` while ` (Para bucles while).
  • ` for-each ` (Bucle para matrices).
  • ` switch ` (Estructura de selección).

5-Operadores logicos


  • ` && ` (Puerta logica "Y").
  • ` || ` (Puerta logica "o").
  • ` ! ` (Negación).

6-Comentarios


  • ` /* */ ` (Para comentarios con multiples líneas).

7-Operadores de conjuntos


  • ` array() ` (Para la creación de matrices).
  • ` => ` (Asignación de valores a claves en matrices).


Sintaxis que utiliza PHP


  • Variables: Las variables en PHP comienzan con el signo $ seguido de un nombre de variable, ademas de que no se necesita declarar el tipo de variable.

    Ejemplo: $numero = 42;
    -------- $nombre = "Juan";

  • Comentarios: Los comentarios de una línea comienzan con //, y los comentarios de múltiples líneas se encierran entre /* y */.

    Ejemplo: // Comentario de una línea
    Ejemplo 2: /* Comentario de múltiples líneas */

  • Estructuras de control: PHP admite estructuras de control como if, else, elseif, while, for, foreach, switch para controlar el flujo del programa.

    Ejemplo:

    if ($condición){
    --- --Condición si la condición es verdadera-----}
    --else{
    --- --Codigo si la condición es falsa-----}

  • Funciones: Puedes definir y llamar funciones en PHP utilizando la palabra clave function.

    Ejemplo:

    function miFuncion($parametro) {
    ---// Código de la función }

  • Operadores de comparación: PHP tiene operadores de comparación, como ==, !=, >, <, >=, <=, para comparar valores.

    Ejemplo:
    $igual = ($a == $b);
    $diferente = ($x != $y);
    $mayor_que = ($m > $n);

  • Tipos de datos: PHP admite varios tipos de datos, como enteros, cadenas, números de punto flotante, booleanos, matrices (arrays), objetos y recursos.

    Ejemplo:

    $entero = 42;
    $cadena = "Hola, mundo";
    $decimal = 3.14;
    $booleano = true;
    $matriz = array(1, 2, 3);

  • Uso de bibliotecas: En PHP, se puede incluir y utilizar bibliotecas externas a través de la función include o require.

    Ejemplo:

    include('mi_biblioteca.php');
    require('mi_biblioteca.php');

Aplicación a descargar para editar y ejecutar

  • Eclipse PDT: Es un entorno de desarrollo integrado ampliamente utilizado de codigo abierto, en el que comprende una gran cantidad de caractersticas que rivalizan a otros entornos de desarrollo, como: la asistencia de codigo, la validación de sintaxis, la gran comunidad que tiene el IDE, el resaltado de sintaxis, etc.

¿A que tipo de paradigma pertenece?


- PHP es un lenguaje de programación cuya sintaxis es heredera de C, con una pequeña diferencia y es que este es un lenguaje de tipo dinamico multiparadigma, de manera que los paradigmas que adopta el lenguaje son: el paradigma imperativo, funcional, orientado a objetos y procedural.


Esqueleto y ejemplo de programa escrito en PHP- Serie de fibonacci



    <?php
    function fib($N) {
        if ($N == 0) {
            return 0;
        } elseif ($N == 1) {
            return 1;
        } else {
            return fib($N - 1) + fib($N - 2);   
        }
    }
                
    $n = 10;
    for ($i = 0; $i < $n; $i++) {
        echo fib($i) . "\n";
    }
    ?>
